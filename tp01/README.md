# CRUD Swing - Projeto de AEDS2 TP01

Este é um projeto de CRUD (Create, Read, Update, Delete) desenvolvido em Java com interface gráfica usando Swing. O objetivo é gerenciar músicas em um banco de dados.

## Funcionalidades

- Adicionar nova música
- Atualizar informações de uma música existente
- Pesquisar música por ID
- Deletar música por ID
- Listar todos os registros

## Como usar

### Execução do programa:

1. Certifique-se de ter o Java Development Kit (JDK) instalado na sua máquina.
2. Compile e execute o arquivo `InterfaceMenu.java` para iniciar a interface gráfica.

### Adicionar nova música:

1. Preencha os campos de texto com as informações da música.
2. Clique no botão "Adicionar" para inserir a música no banco de dados.

### Atualizar música existente:

1. Insira o ID da música que deseja atualizar.
2. Preencha os campos de texto com as novas informações da música.
3. Clique no botão "Atualizar" para salvar as alterações.

### Pesquisar música por ID:

1. Digite o ID da música que deseja pesquisar.
2. Clique no botão "Pesquisar" para encontrar a música correspondente.

### Deletar música por ID:

1. Informe o ID da música que deseja excluir.
2. Clique no botão "Deletar" para remover a música do banco de dados.

### Listar todos os registros:

- Clique no botão "Listar Registros" para visualizar todas as músicas cadastradas.

## Requisitos

- Java Development Kit (JDK) versão 8 ou superior.
- Ambiente de desenvolvimento Java (Eclipse, IntelliJ, etc.) ou compilador Java (Javac).
- Sistema operacional compatível com Java.
